Lazy Shop

Esta é uma página de testes para demonstrar a melhora na pontuação do Google Speed Insigths que é possível, utilizando IntersectionObserver e a biblioteca QuickLink

Em termos de desenvolvimento, estas melhoras são melhor percebidas utilizando as Ferramentas do Desenvolvedor em um navegador para desktop, emulando um celular, para visualizar o carregamento de images com lazy loading.

Também se recomenda desabilitar o cache, na aba network, no painél de Ferramentas do Desenvolvedor.

Referências

  • MDN web docs: Intersection Observer API
  • Google Developers: IntersectionObserver’s Coming into View
  • Github: Google QuickLink
